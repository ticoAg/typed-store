# Todo Index

本目录用于维护主题级研发进度，目标是让任何人或 agent 在中断后都能快速回答：

- 现在在做什么
- 做到哪一步了
- 下一步是什么
- 证据在哪里

## Status Buckets

- `proposed`：已提出，尚未进入执行
- `active`：正在推进
- `blocked`：被阻塞，等待条件满足
- `done`：已完成主要目标
- `archived`：已归档，不再活跃维护

## Recommended Convention

每个持续推进的主题，建立一份：

- `todo/<topic>.md`

建议在文件头说明：

- Status
- Spec
- Code SSOT
- Last Updated
